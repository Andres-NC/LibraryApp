\subsection{IU01 Pantalla de Inicio}

\subsubsection{Objetivo}
	Pantalla de Inicio, en esta Interfaz se podrá generar las citas de entrevista e ir a la Interfaz para consultar entrevistas.  

\subsubsection{Diseño}
	Esta pantalla aparece al iniciar el sistema.  \\\\
	Si en la base de datos no algún registro de citas agendadas, la parte de consultar estará inhabilitada. 

\IUfig[.7]{gui/UI01PantallaInicio}{UI01}{Pantalla de Inicio.}

\subsubsection{Salidas}
	\begin{Citemize}
		\item Ninguna. 
	\end{Citemize}
	
\subsubsection{Entradas}
	\begin{Citemize}
		\item BD Vacio: Variable booleana que indica si exite almenos algún registro en la BD de la entidad entrevistas. Si el valor es 1: habilita el \IUbutton{Consultar} en caso contrario lo deshabilita. 
	\end{Citemize}

\subsubsection{Comandos}
\begin{itemize}
	\item \IUbutton{Generar}: Activa el algoritmo que genera las citas de entrevista de acuerdo a la disponibilidad de horarios y entrevistadores. 
	\item \IUbutton{Consultar}: Muestra la interfaz \IUref{UI02}{Pantalla de Consulta}.
\end{itemize}

\subsubsection{Mensajes}
	\begin{Citemize}
		\item {\bf [E1]} Error al conectar a la BD.
		\item {\bf [E2]} Error al generar las citas de entrevista.
		\item {\bf MSG1} No se ha encontrado registros de entrevistas agendadas previamente. Para consultar primero genere entrevistas. 
	\end{Citemize}
